Supabase Setup — Connect Your Database

1
Go to supabase.com, sign up free, and create a new project. Name it shc-library.
2
In your Supabase project, go to SQL Editor and run this query to create the table:

create table visitor_log (id bigint generated always as identity primary key, name text, id_number text, course text, year_level text, purpose text, resource text, visited_at timestamptz default now());
3
Go to Project Settings → API. Copy your Project URL and anon public key.
4
Also in Authentication → Policies, run this to allow public inserts:

create policy "allow insert" on visitor_log for insert to anon with check (true); create policy "allow select" on visitor_log for select to anon using (true); alter table visitor_log enable row level security;
5
Paste your Project URL and anon key below, then click Save. Your database is live!
